Stumble guys is a multiplayer online game which has multiple game modes that a player has to play in order to win. Each game has three rounds in different game mode. After each round there are a certain number of players who are eliminated. There are a total of 32 players in a game. After the completion of the first round only the first 16 get selected. In the second round only the first 8 will get selected for the final round. In the final round only the player who stands first wins.
